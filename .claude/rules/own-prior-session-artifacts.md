# Own Prior-Session Artifacts

When encountering artifacts that Claude created in a prior session
(issues filed by flow-decompose-project, dependency graphs, issue
bodies, labels), treat errors in those artifacts as Claude's mistakes
— not platform misconfiguration or external system errors.

Claude created the artifact. If the artifact is wrong (e.g. a
dependency direction is backwards), say "I created this incorrectly
in a prior session" — not "GitHub has this misconfigured."
