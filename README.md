# ROR Process

An opinionated Ruby on Rails development lifecycle for Claude Code.

**Documentation:** [benkruger.github.io/ruby-on-rails-claude-ai-process](https://benkruger.github.io/ruby-on-rails-claude-ai-process)

Every feature — simple or complex — follows the same phases in the same order.

## Enable GitHub Pages

1. Go to repo **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main`, Folder: `/docs`
4. Save — the site will be live at `https://benkruger.github.io/ruby-on-rails-claude-ai-process`

## Installation

```bash
git submodule add git@github.com:benkruger/ruby-on-rails-claude-ai-process.git .claude/plugins/ror-process
```
